## How to import csv files using sqlite command line :

Open sqlite3 command line :
- sqlite3

To import '.csv' file, from sqlite command line, run :
- .import FILENAME.csv FILENAME

To run a file with extension '.sql', from sqlite command line, run :
- .read FILENAME.sql

To view the entire tables available, from sqlite command line, run :
- .tables

To clair the sqlite command line, run : 
- .shell cls
